# 포트폴리오

안녕하세요 노주훈의 포트폴리오 사이트에 오신 것을 환영합니다. 이 사이트는 저의 프론트엔드 개발자로서의 기술과 프로젝트를 소개하는 공간입니다.

## 개요

이 포트폴리오 사이트는 **Next.js 14** 버전을 사용하여 제작되었으며, GitHub Actions를 적용하여 Git 배포시 자동으로 GitHub Pages 에 호스팅되고 있습니다. 블로그 게시물, 이전 회사에서 작업한 프로젝트 및 개인적인 프로젝트를 포함한 작업을 한 곳에서 볼 수 있습니다.

## 기술 스택

- **React.js/Next.js**: 포트폴리오의 주요 프레임워크로 최신 버전의 Next.js 사용.
- **Vue.js/Nuxt.js**: 업무 페이지 개발에 사용.
- **React Native**: 토이 프로젝트 모바일 앱 개발에 활용.
- **Jekyll**: 블로그 섹션에 사용.
