How to Change Sidebar Background Color ?

Go to the dist js app.init.js.
Search for the SidebarColor in that.
Now, you can choose a value from skin1 to skin6 as per your need.

How to Change Header Background Color ?

Go to the dist js app.init.js.
Search for the NavbarBg in that.
Now, you can choose a value from skin1 to skin6 as per your need.

How to Change Logo Background Color ?

Go to the dist js app.init.js.
Search for the LogoBg in that.
Now, you can choose a value from skin1 to skin6 as per your need.

How to Change Global Fonts ?

we have used google fonts in our template. so, when you change the fonts of the theme go tho the scss folder and open variable.scss file. In that we have import the google fonts link. 
                    
@import url('https://fonts.googleapis.com/css?family=Rubik:300,400,500,700,900');
                                                    
                                                  
                    
$font-family-sans-serif: "Rubik", sans-serif !default;
If you want to change fonts of the theme then put the URL of your fonts and change font family in the variable.scss.
